1.  安装node.js
    *   下载node.js

        到[node.js](https://nodejs.org/en)的官方网站下载相应的node.js的程序包。

    *   解压缩程序包

            tar zxvf node-v4.1.1-linux-x64.tar.gz
    
    *   将程序放在特定的目录上，并修改环境变量。最终验证node的版本号：
            node -v

        如果显示下面的信息说明部署正确：
        
            v4.1.1

2.  安装gitbook工具

    *   使用npm工具安装
    
            npm install gitbook -g
